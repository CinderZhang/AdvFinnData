Week 6: Dictionaries and Sets in Python

Learning Objectives:

  • Understand the concept of dictionaries and sets in Python.
  • Learn how to create, access, and manipulate dictionaries and sets.
  • Understand the use cases for dictionaries and sets.

Materials:

Exercises:

  1. Write a Python program to combine two dictionaries by adding values for common keys.
  2. Write a Python program to create a dictionary from a string where characters are the keys and the values are the number of occurrences of each character.
  3. Create a Python program that takes a set and returns a new set with unique squared values.
  4. Write a Python program to find the elements that are in one set but not in another.
  5. Write a Python program to create a dictionary with keys as numbers between 1 and 20 and values as the squares of the keys.
  6. Create a Python program that takes a list and returns a dictionary with the list items as keys and the frequencies of each item as values.
