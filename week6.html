Week 6: Working with Financial Data using Pandas

Learning Objectives:

  • Understand the basics of the Pandas library in Python.
  • Learn how to load and manipulate financial data using Pandas.
  • Understand how to perform basic financial analysis using Pandas.
  • Learn how to fetch financial data from APIs using Python.

Exercises:

  1. Load a dataset of stock prices into a Pandas DataFrame and calculate the daily returns.
  2. Calculate the moving average of a stock price over a given window of days.
  3. Use Pandas to merge data from multiple stocks into a single DataFrame and analyze the correlations between their daily returns.
  4. Create a portfolio of stocks and calculate its overall returns and risk using Pandas.
  5. Visualize the historical stock prices and moving averages using Pandas plotting functionality.
  6. Fetch stock data from an API and load it into a Pandas DataFrame. Perform some basic analysis on this data.
