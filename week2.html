Week 2: Python Basics: Variables, Data Types, and Operations

Learning Objectives:

  • Understand variables and data types in Python.
  • Perform basic operations with variables.
  • Learn how to take input and produce output in Python programs.

Materials:

1. Variables in Python

What are variables? How to declare and assign values to variables in Python.

Read more about Variables in Python

2. Data Types in Python

Understanding different data types in Python: integers, floats, strings, booleans.

Read more about Data Types in Python

3. Basic Operations

Performing arithmetic and logical operations with variables in Python.

Read more about Basic Operations in Python

4. Input and Output in Python

How to take input from the user and display output in Python programs.

Read more about Input and Output in Python

Exercises:

  1. Create variables of different data types and perform operations on them.
  2. Write a Python program that takes input from the user and displays it.
  3. Practice type conversion by taking input as a string and converting it to an integer or float to perform arithmetic operations.
  4. Write a Python program to calculate the area of a rectangle. Take the length and width as inputs from the user.
  5. Create a simple calculator that can add, subtract, multiply, and divide two numbers entered by the user.
  6. Write a Python program that takes a temperature in Celsius and converts it to Fahrenheit.
  7. Write a Python program that reverses a string entered by the user.
