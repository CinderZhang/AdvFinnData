Week 4: Functions in Python

Learning Objectives:

  • Understand the concept of functions in programming.
  • Learn how to define and call functions in Python.
  • Understand the importance of code reusability and modular programming.

Materials:

1. Defining and Calling Functions

Learn how to define functions and call them in your Python programs.

Read more about Defining and Calling Functions in Python

2. Function Parameters and Arguments

Understand how to pass parameters and arguments to functions.

Read more about Function Parameters and Arguments in Python

3. Return Values

Learn how functions can return values that can be used in your program.

Read more about Return Values in Python

Exercises:

  1. Write a Python function that takes two numbers as parameters and returns their sum.
  2. Write a Python function that takes a list of numbers and returns the maximum value.
  3. Write a Python function that takes a string and returns the number of words in the string.
  4. Create a Python function that calculates the area of a circle given its radius.
  5. Write a Python function that takes an integer n and returns the nth Fibonacci number.
  6. Create a Python function that takes a list of strings and returns a list of their lengths.
