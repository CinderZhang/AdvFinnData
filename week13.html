Week 13: Capstone Project

Learning Objectives:

  • Apply the skills and knowledge acquired throughout the course in a real-world project.
  • Develop a financial data analysis application using Python.
  • Present the project and receive feedback.

Capstone Project:

For the final week, you will work on a capstone project that involves creating a financial data analysis application using Python. This project should incorporate the various concepts and skills learned throughout the course, including data manipulation, visualization, web scraping, API usage, and machine learning.

Project Requirements:

  • Use Python for data analysis and manipulation.
  • Include data visualization components.
  • Implement web scraping or API integration to gather financial data.
  • Apply machine learning algorithms for predictive analysis.
  • Document the code and write a report explaining the methodology and results.
  • Present the project to peers and instructors.

Resources:

Python Official Website
Pandas Documentation
Matplotlib Documentation
Scikit-learn Documentation

Submission:

Submit your code, documentation, and presentation slides before the deadline. Ensure that you have tested the application thoroughly and that the documentation is clear and concise.
