Week 7: File Handling and Exception Handling in Python

Learning Objectives:

  • Understand how to read from and write to files in Python.
  • Learn how to handle exceptions and errors in Python programs.
  • Understand the importance of exception handling in building robust applications.

Materials:

1. File Handling in Python

Learn how to open, read, write, and close files in Python.

Read more about File Handling in Python

2. Exception Handling in Python

Learn how to handle exceptions using try, except, finally blocks and how to raise exceptions.

Read more about Exception Handling in Python

Exercises:

  1. Write a Python program to read a text file and count the number of lines in the file.
  2. Write a Python program to write a list of random numbers to a file and then read the file to display the numbers.
  3. Create a Python program that takes a file name as input and prints the contents of the file. Handle the exception if the file does not exist.
  4. Write a Python program that takes two numbers as input and divides them. Handle any exceptions that may occur during the division process.
  5. Write a Python program to copy the contents of one file to another file.
  6. Create a Python program that reads a file and calculates the average of the numbers in the file. Handle exceptions for file not found and division by zero.
