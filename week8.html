Week 8: Modules and Packages in Python

Learning Objectives:

  • Understand the concept of modules in Python.
  • Learn how to create and use modules.
  • Understand the concept of packages in Python.
  • Learn how to create and use packages.

Materials:

1. Modules in Python

Learn how to create and import modules in Python.

Read more about Modules in Python

2. Packages in Python

Learn how to create and use packages in Python.

Read more about Packages in Python

Exercises:

  1. Write a Python module that contains functions for calculating the area and perimeter of a rectangle. Import this module into another program and use the functions.
  2. Create a Python package that contains modules for handling different shapes (e.g., circles, rectangles, triangles). Write a program that imports this package and uses it to perform geometric calculations.
  3. Write a Python program that imports a module and prints the list of all functions and variables defined in the module.
  4. Create a Python package with a module that contains a function to calculate the factorial of a number. Write a script that imports this package and uses the function.
