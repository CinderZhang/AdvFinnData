1. Time Series Fundamentals

Understanding the basic components and characteristics of time series data.

import pandas as pd
import numpy as np
from statsmodels.tsa.seasonal import seasonal_decompose

# Load and process time series data
df = pd.read_csv('stock_data.csv', index_col='Date', parse_dates=True)
result = seasonal_decompose(df['Close'], period=252)  # 252 trading days

# Plot components
result.plot()

2. Moving Averages and Technical Analysis

Learn how to calculate and interpret moving averages for technical analysis.

# Calculate moving averages
df['SMA_50'] = df['Close'].rolling(window=50).mean()
df['EMA_20'] = df['Close'].ewm(span=20).mean()

# Generate trading signals
df['Signal'] = np.where(df['SMA_50'] > df['EMA_20'], 1, -1)

3. Time Series Forecasting

Introduction to forecasting methods using statistical models.

from statsmodels.tsa.holtwinters import ExponentialSmoothing

# Fit Holt-Winters model
model = ExponentialSmoothing(df['Close'],
                           seasonal_periods=252,
                           trend='add',
                           seasonal='add')
fitted_model = model.fit()

# Make predictions
forecast = fitted_model.forecast(30)  # 30-day forecast
