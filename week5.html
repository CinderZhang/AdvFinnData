Week 5: Lists and Tuples in Python

Learning Objectives:

  • Understand the concept of lists and tuples in Python.
  • Learn how to create, access, and manipulate lists and tuples.
  • Understand the difference between lists and tuples.

Exercises:

  1. Write a Python program to find the largest number in a list.
  2. Write a Python program to count the number of strings in a list where the string length is 2 or more.
  3. Write a Python program to remove duplicates from a list.
  4. Create a Python program that takes a tuple of numbers and computes the sum of all the numbers in the tuple.
  5. Write a Python program to convert a list of tuples into a dictionary.
  6. Create a Python program that takes two lists and returns a list that contains only the elements that are common between the two lists (without duplicates).
  7. Write a Python program to find the second smallest number in a list.
  8. Write a Python program that takes a list of numbers and returns a new list with unique elements of the first list.
